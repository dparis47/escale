# Commande /nouvelle-session

Tu démarres une nouvelle session sur le projet L'Escale.

1. Lis le fichier CLAUDE.md pour retrouver le contexte complet
2. Explore la structure du projet (dossiers et fichiers existants)
3. Lis le schéma Prisma si disponible (prisma/schema.prisma)
4. Résume en quelques lignes : ce qui a été fait, ce qui est en cours, ce qui reste à faire
5. Demande à l'utilisateur ce qu'il veut faire

Ne commence aucun développement avant que l'utilisateur te l'ait demandé.
